## 功能

- **抽单人/抽多人/抽小组 等抽取方式**
- **有着结果语音提示的功能**
- **可开关的历史记录功能**
- **历史记录还可以查看每个人被抽取的时间、方式**
- **甚至还可以设置与切换多个班级的名单**
- **抽取界面与设置界面的隔离**
- **方便快捷的悬浮窗设计**
- **有着开机自启动的便捷**
- **有设置界面的密码功能（请期待后续更新）**
- **现两种不同本质的权重方式（请期待后续更新）**
- **现代化的设计界面**
